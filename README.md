# 📱 iPhone Website Clone

This project is a visually appealing and animated clone of Apple's iPhone web page, built using **React**, **GSAP**, **Tailwind CSS**, and **Vite**. It demonstrates smooth scrolling animations, interactive transitions, and responsive design — aiming to closely match the official Apple aesthetic.

---

## 🚀 Tech Stack

- ⚛️ **React** – Component-based UI library
- ⚡ **Vite** – Fast development server and bundler
- 🎨 **Tailwind CSS** – Utility-first CSS framework
- 🎞️ **GSAP** – High-performance animations

---

## 📂 Features

- ✅ Smooth scroll-triggered animations with GSAP
- ✅ Responsive layout across devices
- ✅ Pixel-perfect styling using Tailwind CSS
- ✅ Modular and clean component structure
